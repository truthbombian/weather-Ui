# Weather UI Website

A simple weather application that displays current weather information for a given city.

## Features

- Search for weather by city name
- Displays temperature, weather description, feels like, humidity, and wind speed
- Responsive design with a clean UI

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Node.js with Express for serving static files
- Weather data from wttr.in API

## Installation

1. Clone or download the repository
2. Navigate to the project directory
3. Run `npm install` to install dependencies

## Usage

1. Start the server: `npm start`
2. Open your browser and go to `http://localhost:3000`
3. Enter a city name in the search box and click "Search" or press Enter
4. View the current weather information

## API

The app uses the wttr.in API for weather data, which provides free weather information without requiring an API key.

## Troubleshooting

- Make sure Node.js and npm are installed
- Ensure port 3000 is not in use by another application
- Check your internet connection for API calls
